Origens do HTML

09/04/2024

Contexto histórico: O HTML teve suas raízes no final dos anos 80 e início dos anos 90, quando Tim Berners-Lee e sua equipe no CERN (Organização Europeia para a Pesquisa Nuclear) desenvolveram o conceito de hipertexto para compartilhar documentos científicos online.

Propósito inicial: O HTML foi concebido como uma linguagem de marcação simples para criar e compartilhar documentos hipertexto na World Wide Web em ascensão.

HTML 1.0

09/04/2024

Lançamento: A primeira versão oficial do HTML, HTML 1.0, foi lançada em 1993.

Recursos básicos: Incluía recursos básicos de marcação para estruturar documentos online, como títulos, parágrafos, listas e links.

Limitações: O HTML 1.0 era bastante rudimentar e não suportava muitos dos recursos que consideramos essenciais hoje, como imagens embutidas ou tabelas.

HTML 2.0

09/04/2024

Lançamento: HTML 2.0 foi lançado em 1995.

Expansão de recursos: Introduziu recursos adicionais, como formulários e tabelas, permitindo uma maior variedade de layouts e interatividade.

HTML 3.2

09/04/2024

Lançamento: HTML 3.2 foi lançado em 1997.

Introdução de recursos avançados: Incluiu suporte para frames, imagem de fundo e applets de Java, expandindo ainda mais as capacidades do HTML.

HTML 4.01

09/04/2024

Lançamento: HTML 4.01 foi lançado em 1999.

Melhorias significativas: Introduziu melhorias na acessibilidade, formulários e scripts, e também foi a primeira versão a incluir o conceito de estilo em cascata (CSS).

XHTML (Extensible Hypertext Markup Language)

09/04/2024

Conceito e adoção: Introduzido no final dos anos 90, o XHTML foi uma reformulação do HTML como uma aplicação do XML (Extensible Markup Language), seguindo regras mais rígidas de sintaxe e estrutura.

Racionalidade: XHTML foi desenvolvido para tornar os documentos HTML mais compatíveis com o XML e, portanto, mais facilmente processáveis por programas de software.

HTML5

09/04/2024

Lançamento: HTML5 foi lançado em 2014.

Nova era no desenvolvimento web: Introduziu novos elementos semânticos, APIs para multimídia, gráficos vetoriais, armazenamento local e muito mais, representando uma grande evolução no desenvolvimento web.

Suporte a dispositivos móveis: HTML5 trouxe recursos que facilitaram a criação de sites responsivos e aplicativos web para dispositivos móveis.

Web Components

09/04/2024

Conceito: Os Web Components são uma tecnologia que permite aos desenvolvedores criar seus próprios elementos HTML personalizados e reutilizáveis, encapsulando HTML, CSS e JavaScript em um único pacote.

HTML Semântico

09/04/2024

Significado e importância: O HTML semântico é uma abordagem no desenvolvimento web que enfatiza o uso de tags HTML de maneira semântica para fornecer significado estrutural e de conteúdo ao conteúdo da página.

Benefícios: Melhora a acessibilidade, a indexação pelos motores de busca e a compreensão do conteúdo por humanos e máquinas.
